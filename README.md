# Marathon Skills Web

Браузерная версия приложения Marathon Skills без серверной части.

## Возможности

- регистрация и вход аккаунтов бегунов;
- административный аккаунт `admin / admin123`;
- выход из аккаунта с очисткой текущего черновика;
- регистрация участников с проверкой введенных данных;
- список стран, маска телефона и проверка возраста;
- загрузка, перемещение и кадрирование фотографии;
- интерактивный расчет BMI;
- список участников;
- удаление участников администратором;
- сохранение данных в `localStorage`.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```powershell
py -m http.server 4173 --directory .
```

После этого откройте `http://127.0.0.1:4173`.

## Технологии

- HTML5;
- CSS3;
- JavaScript;
- Web Crypto API;
- Canvas API;
- Local Storage API.
